# Testing Scripts — Master Instructional Design Skill

A collection of test prompts for evaluating the `master-instructional-design` skill across its core modes and capabilities.

---

## 1. Audit Mode

**Test: Learning Objective Audit**
```
Audit this learning objective: "Learners will understand the importance of data privacy in the workplace."
```

**Test: Storyboard Critique**
```
Here's a storyboard slide:
- Title: "What is HIPAA?"
- Content: A bulleted list of 12 HIPAA facts
- Interaction: Click Next
- Narration: Reads the bullet points verbatim

Audit this slide.
```

**Test: Assessment Item Review**
```
Review this quiz question:
"Which of the following is NOT a characteristic of adult learners according to Knowles?"
A) Self-directed
B) Experience-based
C) Passive recipients of information
D) Problem-centered

Is this a strong assessment item?
```

---

## 2. Process Coach Mode

**Test: Needs Analysis Coaching**
```
My VP just asked me to build a 2-hour eLearning course on "communication skills" for our sales team. Where do I start?
```

**Test: Root Cause Diagnosis**
```
Our customer service reps keep giving wrong refund amounts. Leadership wants a training course. What should I do first?
```

**Test: Solution Framing**
```
I've completed a needs analysis for a new employee onboarding program. The root cause is a knowledge/skill gap — new hires don't know our internal systems or processes. The audience is 50–100 new hires per quarter, all remote, varying tech comfort levels. What modality do you recommend and why?
```

---

## 3. Theory & Concepts Mode

**Test: Framework Comparison**
```
What's the difference between ADDIE and SAM? When would you choose one over the other?
```

**Test: Cognitive Load Theory Application**
```
Explain Sweller's Cognitive Load Theory and give me three concrete ways to apply it in an eLearning module on financial compliance.
```

**Test: Academic Deep Dive**
```
Compare Dick & Carey's Systematic Design Model with Smith & Ragan's Instructional Design model. Where do they align and where do they diverge?
```

---

## 4. Co-Creation Mode

**Test: Objective Writing**
```
Write three strong Bloom's-aligned learning objectives for a module on conducting performance reviews. One each at the Apply, Analyze, and Evaluate levels.
```

**Test: Scenario Writing**
```
Write a branching scenario for a module on handling customer complaints. The learner plays a customer service rep. Include one correct path and two consequence paths.
```

**Test: Formative Assessment Design**
```
I'm building a 20-minute eLearning module on project risk management for mid-level PMs. Design a formative assessment architecture for the module.
```

---

## 5. Authoring Tool Expert Mode

**Test: Storyline Technical Guidance**
```
In Articulate Storyline 360, how do I create a variable that tracks how many times a learner has visited a slide, and use it to show different feedback after the third visit?
```

**Test: JavaScript in Storyline**
```
Write a JavaScript snippet for Storyline 360 that disables the Next button until the learner has clicked all hotspots on a slide.
```

**Test: SCORM Publishing**
```
What SCORM 2004 publishing settings should I use in Storyline 360 to ensure completion tracking works correctly in a Cornerstone LMS?
```

---

## 6. Inclusive & Emotional Design Mode

**Test: Stereotype Threat Audit**
```
I'm building a cybersecurity compliance course. The opening scenario features "Bob, a busy executive who clicked a phishing link." Is there anything I should reconsider about this framing?
```

**Test: Psychological Safety Design**
```
I'm designing a DEI training on unconscious bias for a manufacturing company. Leadership wants to open with a self-assessment that asks employees to rate their own biases. What are the risks, and how would you redesign the opening?
```

**Test: Emotional Arc Design**
```
Map the emotional arc for a mandatory compliance training on workplace harassment prevention. What should learners feel at the opening, middle, and close — and what design choices produce those emotions?
```

---

## 7. Agile & Scrum for L&D Mode

**Test: Backlog Structure**
```
I'm using Scrum to manage a large onboarding program with 8 modules. How do I structure this as a product backlog with epics, user stories, and acceptance criteria?
```

**Test: Sprint Planning for ID**
```
My team has a 2-week sprint. We're designing Module 3 of a leadership development program. It needs a storyboard, SME review, and revision cycle. How do I plan this sprint realistically?
```

---

## 8. Generative AI for L&D Mode

**Test: Prompt Engineering**
```
Write an optimized prompt for using an LLM to generate a realistic branching scenario for a module on ethical decision-making in procurement. Include role, context, constraints, and output format in the prompt.
```

**Test: Responsible AI Use**
```
My organization wants to use AI to generate all course content going forward to cut costs. What are the risks I should raise with leadership, and how would I build an AI-assisted workflow that maintains quality?
```

## 9. CLO & Learning Leadership Mode

**Test: L&D Business Case**
```
My CEO doesn't see L&D as a strategic function — she sees it as a cost center. How do I build a business case for a $500K learning technology investment?
```

**Test: Learning Culture Strategy**
```
Our organization scores in the bottom quartile on "learning culture" in our engagement survey. What are the 3–4 highest-leverage interventions a CLO should prioritize?
```

## 10. Edge Cases & Calibration Tests

**Test: Beginner User**
```
I just got hired as an instructional designer and my manager asked me to write learning objectives. I have no idea where to start. Help?
```

**Test: Expert User**
```
I'm weighing whether to use van Merriënboer's 4C/ID model vs. Merrill's First Principles for a complex skill-based curriculum on surgical team communication. What are the tradeoffs?
```

**Test: Resistant Stakeholder Scenario**
```
My stakeholder insists the course needs to be 4 hours long with 60 slides because "that's how we've always done it." How do I handle this?
```

**Test: Non-Training Solution**
```
Our warehouse team keeps making picking errors. The manager wants a training course. I suspect the picking software interface is the real problem. How do I handle this?
```

**Test: Scope Boundary**
```
Can you help me write a Python script to scrape LMS data?
```
> Expected: Skill should redirect to L&D expertise; this is outside its scope.

---

## Evaluation Criteria

For each test, assess the skill response against:

| Criterion | Description |
|---|---|
| **Accuracy** | Is the content factually and theoretically correct? |
| **Relevance** | Does the response address what was actually asked? |
| **Depth calibration** | Is the depth appropriate to the user's apparent level? |
| **Inclusive lens** | Does the response apply DEI and emotional design considerations where relevant? |
| **Actionability** | Does the response give the user something concrete to do next? |
| **Tone** | Warm, direct, developmental — not condescending or generic? |
| **Framework citation** | Are frameworks named and applied, not just referenced? |
| **Coaching posture** | Does the response coach capability, not just dispense answers? |
